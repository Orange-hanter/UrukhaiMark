# Глоссарий

| Термин | Расшифровка | Комментарий |
|--------|-------------|-------------|
| **ГИС «Электронный знак»** | Гос. ИС маркировки РБ | Оператор: РУП «Белбланкавыд», [datamark.by](https://datamark.by/) |
| **ГИС МТ / «Честный знак»** | Гос. ИС маркировки РФ | Оператор: ЦРПТ, [markirovka.ru](https://markirovka.ru/) |
| **КМ** | Код маркировки | GTIN + serial + криптохвост + GS-разделители |
| **КИ** | Код идентификации | КМ без криптохвоста, 31 символ |
| **СИ** | Средство идентификации | КМ в виде GS1 DataMatrix |
| **УКЗ** | Унифицированный контрольный знак | Физический бланк, не DataMatrix на товаре |
| **ЗМН** | Защищённый материальный носитель | Бланк с нанесённым СИ |
| **ЗЗ** | Знак защиты | Защита этикетки от подделки |
| **GTIN** | Global Trade Item Number | 14 цифр |
| **GLN** | Global Location Number | Обязателен в «Электронном знаке» |
| **GCP** | Global Company Prefix | Префикс GS1; без него нет GTIN |
| **ePASS** | Банк электронных паспортов | [epass.by](https://epass.by/) |
| **label_type** | Тип кода в API | 7 = экспорт РФ, 20 = СИ РБ |
| **FNC1** | Function Code 1 | ASCII 232 — первый символ GS1 DataMatrix |
| **GS** | Group Separator | ASCII 29 (`\u001d`) |
| **CRPT** | Оператор «Честного знака» РФ | True API |
| **True API** | REST API ГИС МТ РФ | [docs.crpt.ru](https://docs.crpt.ru/gismt/True_API/) |

**QR vs DataMatrix:** в быту говорят «QR Честного знака», но стандарт — **GS1 DataMatrix** (ECC 200). Форматы несовместимы.
