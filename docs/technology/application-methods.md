# Способы нанесения кодов маркировки

Обзор технологий физического нанесения СИ (GS1 DataMatrix) и УКЗ на товар.

> **Промышленное нанесение на упаковку** (inline CIJ, лазер, VDP на линии, print-and-apply) — отдельный документ: **[industrial-marking.md](industrial-marking.md)**.

## Сводная таблица методов

| Метод | СИ | УКЗ | Объём | Уровень |
|-------|-----|-----|-------|---------|
| [Промышленное inline](industrial-marking.md) | Да | — | 5k+/день | **Завод / линия** |
| Термоэтикетка настольная | Да | — | до 10k/день | Пилот / малый цех |
| УКЗ-бланк | — | Да | любой | Пиво РБ |

---

## Настольные этикетки (кратко)

Для пилота и малых объёмов. **Не** промышленный стандарт при росте линии — см. [industrial-marking.md](industrial-marking.md).

**Схема:** КМ → encoder → термопринтер → ручная/полуавтомат наклейка.

| Когда достаточно | Когда переходить на inline |
|------------------|----------------------------|
| до 3–5 тыс. шт/день | > 50 шт/мин на линии |
| нет конвейера | нужен 100% verify + reject |
| MVP / первые отгрузки | CAPEX окупается за счёт FTE |

Термотрансфер + resin ribbon для аэрозолей. Контроль «первые три» — [quality-control.md](quality-control.md).

---

## УКЗ (пиво, РБ)

Физический **бланк знака** — не DataMatrix. Заказ через API УКЗ, наклейка в зоне по инструкции. Автоматизация: промышленные аппликаторы на коробах.

См. [domestic-rb-beer.md](../processes/domestic-rb-beer.md), [industrial-marking.md § print-and-apply](industrial-marking.md#4-print-and-apply-промышленный).

## Выбор метода для целевых SKU

## Чеклист перед запуском

- [ ] Тестовые КМ из sandbox на **реальной** упаковке
- [ ] Grade ≥ C на целевом оборудовании
- [ ] Для линии: reject + waste flow для брака
- [ ] addMark только после физической маркировки
